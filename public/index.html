Upload Guidelines

Video Format

MP4 file format (H.264 codec recommended). Max file size is 500MB.

Aspect Ratio & Resolution

Vertical layout (9:16 aspect ratio) is highly recommended. Resolution should be at least 720x1280.

Scheduled Posting

Posts will remain queued on this server. Please ensure the computer hosting this script is running at the scheduled time.

TikTok Sandbox

If your developer app is in Sandbox mode, you can only publish to the developer account, and the video's default status will be Private.

Developer Configuration

Must exactly match the Redirect URI configured in the TikTok Developer Center.

Setup Instructions

  1. Register Developer Account

    Go to TikTok for Developers and create an account.

  2. Create Developer App

    Register a new application. Choose **Web App** as platform type.

  3. Request Scope Access

    In App settings, request access to the **Content Posting API**. Specifically ensure the `video.publish` scope is requested.

  4. Set Redirect URI

    Under the App's configuration settings, register: `http://localhost:3000/callback` in the **Redirect URI** list.

  5. Authenticate Account

    Once keys are saved above, click the **Connect TikTok Account** button that appears on the sidebar or dashboard to complete OAuth consent.
